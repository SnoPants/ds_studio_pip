DS Studio – Maya Technical Animation Toolkit

A modular Python-based rigging and pipeline framework for Autodesk Maya, designed to support scalable tool development and reusable rig systems.

Overview

DS Studio is a personal technical animation framework built to:

  • Auto-load at Maya startup
  • Provide reusable rigging components
  • Support modular tool expansion
  • Encourage clean pipeline architecture

This project will serve both a production-ready toolkit and an ongoing learning platform for developing scalable technical systems.

Goals

  • Build a structured rigging library
  • Create modular rig builders (Biped, Limbs, etc.)
  • Integrate cleanly with Unreal Engine export workflows
  • Expand toward UI tools and automation systems

Status

🚧 Work in Progress This framework is actively evolving as I refine architecture, module structure, and pipeline scalability.
